# public/

Drop these two files here before deploying:

1. **retro-pc.png** — your voxel PC hero image (used by src/components/voxel/VoxelPC.jsx).
   If absent, an SVG fallback renders automatically — no errors.

2. **Akhilesh_Kotegar_Resume.pdf** — served by the `Download Resume`
   button and the `/download-cv` CLI command.

Anything else placed here is served as a static asset by Vite.
